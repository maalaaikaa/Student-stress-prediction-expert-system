# 🎓 Student Stress Prediction Expert System

An AI-powered desktop application that predicts students' stress levels using a rule-based expert system and a Forward Chaining inference engine. The system allows students to select stress-related symptoms and provides stress assessment results along with personalized recommendations and coping tips.

---

## 📌 Project Overview

The Student Stress Prediction Expert System is designed to identify and predict a student's stress level based on different academic, lifestyle, psychological, social, and organizational factors.

The application uses Artificial Intelligence concepts, specifically an Expert System and Forward Chaining reasoning technique, to simulate human expert decision-making and provide meaningful stress assessments.

## ⚙️ System Workflow

1. User launches the application.
2. User selects symptoms from different categories.
3. The system collects the selected symptoms as facts.
4. The Forward Chaining engine evaluates the facts against predefined rules.
5. Appropriate rules are fired.
6. The system determines the stress level:

   * Low Stress
   * Moderate Stress
   * High Stress
   * Severe Stress
7. Personalized suggestions and tips are displayed.
8. Assessment history is saved during the session.
